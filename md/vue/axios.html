    axios HTTP库

    qinhaoqvq Vue

    # 是什么？

    axios 是一个基于 promise 的 HTTP 库，可以用在浏览器和 node.js 中。是前端流行的 ajax 请求框架，它是 React 和 Vue 推荐使用的

    # 能干嘛？

    • 通过浏览器创建 XMLHttpRequest 对象，发送 ajax 请求
    • 从 node.js 创建 http 请求
    • 支持 Promise API
    • 支持拦截请求和响应数据
    • 支持取消请求
    • 客户端支持防御 XSRF

    # 怎么用？

    # 发送请求

    1. 调用函数：axios({config})

    2. 对象方法：axios.request(config)或axios.get(url,config)或axios.post(url,data,config)

    3. 通过创建 axios 对象，然后调用方法

      const instance = axios.create({
  baseURL: 'https://some-domain.com/api/',
  timeout: 1000,
  headers: {'X-Custom-Header': 'foobar'}
});
instance.get('path',config)
instance.post('path',config)

    # 接收响应

    响应数据包括以下 6 部分

    1. data：请求服务器响应的数据
    2. status：响应 HTTP 状态码
    3. statusText：响应 HTTP 状态信息
    4. headers：响应头信息
    5. config：请求配置信息
    6. request：（ajax）XMLHttpRequest 实例对象或（node.js）ClientRequest 实例对象

    # 创建 axios 实例对象

    const instance = axios.create({
  baseURL: 'https://some-domain.com/api/',
  timeout: 1000,
  headers: {'X-Custom-Header': 'foobar'}
});

    # 请求配置对象

    1. url
    2. method
    3. baseURL
    4. headers
    5. params：是即将与请求一起发送的 URL 参数，必须是 plain 对象或 URLSearchParams 对象
    6. data：是请求主体的数据对象
    7. timeout：请求超时时间
    8. withCredentials：表示跨域时是否需要使用凭证
    9. responseType：服务器响应数据类型
    10. responseEncoding：服务器响应数据编码格式
    11. maxContentLength：指定最大的响应内容长度
    12. cancelToken：指定用于取消请求的 cancle token
    13. proxy：

    请求配置对象可以通过全局的 axios.defaults 默认值属性、lib/defaults.js文件和请求时传入的config参数三个地方进行配置，配置会以一个优先顺序进行合并。优先级：config > defaults > defaults.js

    # 拦截器

    # 添加拦截器

    1. const i_req = axios.interceptors.request.use(function(config){},function(error){})
    2. const i_res = axios.interceptors.response.use(function(response){},function(error){})

    # 取消拦截器

    1. axios.interceptors.request.eject(i_req)
    2. axios.interceptors.response.eject(i_res)

    # 取消请求

    const CancelToken = axios.CancelToken;
let cancel;

axios.get('/user/query', {
  cancelToken: new CancelToken(function executor(c) {
    // executor 函数接收一个 cancel 函数作为参数
    cancel = c;
  })
});

// cancel the request
cancel();
