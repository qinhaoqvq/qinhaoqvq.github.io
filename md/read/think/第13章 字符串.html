    # String对象

    不变行，String对象都是有一个final char[]数组实现的，因此String对象的值是不可变得。任何看起来改变String对象值的行为，其实质就是创建了一个新的String对象

    # ✨重载 +与StringBuilder

    使用+进行连接字符串的操作，其底层是编译器自动使用StringBuilder和append方法来实现的，因为此方法更加高效

    # 代码验证

    public class Concatenation {
    public static void main(String[] args) {
        String mango = "mango";
        String str = "abc" + mango + "def" + 47;
        System.out.println(str);
    }
}
// 通过javap -c反汇编
// 编译器自动引入了StringBuilder类，用以构造最终的String对象，并为每个字符串调用一次的append方// // 法，最后通过toString()生成结果

    # 经验

    ✨如果在for循环中使用字符串拼接操作，在循环中显示的使用StringBuilder对象会更加高效。因为使用+即隐式的使用StringBuilder对象，但是每次for循环编译器都会自动创建一个StringBuilder对象，所以效率比较低

    # 格式化输出

    # Formatter

    Formatter可以看做是一个翻译器，它将你的格式化字符串和数据翻译成需要的结果。当创建一个Formatter对象时，需要向其构造方法中传递一些信息，告诉它将最终结果输出到哪里。默认是输出到new StringBuilder()

    # String.format()

    String.format()底层实现就是Formatter对象。return new Formatter().format(format, args).toString();

    # 正则表达式

    # 是什么

    一般来说，正则表达式就是以某种方式来描述字符串

    # 能干嘛

    正则表达式是一种强大而灵活的文本处理工具。使用正则表达式，我们能够以编程的方式，构造复杂的文本模式，并对输入的字符串进行搜素。一旦找到了匹配这些模式的部分，就更加方便的对它进行处理

    # split()

    String类自带了正则表达式工具，它就是split()

    # Pattern

    1. 使用静态方法Pattern.compile()编译正则表达式，并根据输入正则字符串生成一个Pattern对象
    2. 使用Pattern对象的matcher()方法检索你输入的字符串，并生成一个Matcher对象

    # Matcher

    使用Matcher对象的方法，对匹配的部分进行操作

    # 方法

    1. boolean matches()：判断输入字符串能否完全匹配
    2. boolean lookingAt()：判断输入字符串最前面的部分是否能够匹配
    3. boolean find()：判断输入字符串的部分是否能够匹配
    4. replaceAll(String str)：使用该字符串替换匹配到的部分
    5. int start()：匹配到的字符串在原目标字符串中的位置开始位置
    6. int end()：匹配的字符串的最后一个字符在原目标字符串中的索引位置

    # 组

    组是用括号划分的正则表达式，可以根据组的编号来引用某个组。组号为0表示整个表达式，组号1表示被第一对括号括起来的组，以此类推。

    public static void main(String[] args) {
        // 两个组, 1表示（a*b）, 2表示（[0-9]+）
        Pattern pattern = Pattern.compile("(a*b)([0-9]+)");
        Matcher matcher = pattern.matcher("aacab1234xxxxaaaaab4321dog");
        while (matcher.find()) {
            System.out.println(matcher.group());
            System.out.println(matcher.group(0));
            System.out.println(matcher.group(1));
            System.out.println(matcher.group(2));
        }
    }
// 执行结果
ab1234
ab1234
ab
1234
-----------
aaaaab4321
aaaaab4321
aaaaab
4321

    # 扫描输入

    # 是什么

    Scanner负责从文件或者标准输入读取数据，它的构造方法可以接收任何类型的输入对象，包括File对象、InputStream、String或者其他实现Readable接口的对象

    # Scanner定界符

    在默认情况下，Scanner根据空白符对输入进行分词，但是可以使用正则表达式自定义定界符

    scanner.useDelimiter(",")指定使用逗号为定界符

     public static void main(String[] args) {
        // 默认定界符为 空白字符
        Scanner scanner = new Scanner("1 2 3");
        while (scanner.hasNext()) {
            System.out.println(scanner.next());
        }
        // 修改定界符为 ,
        Scanner point = new Scanner("1,2,3");
        point.useDelimiter(",");
        while (point.hasNext()) {
            System.out.println(point.next());
        }
    }

    # StringTokenizer

    在JDK5之前，使用StringTokenizer来分割字符串，该类已经废弃不再使用
